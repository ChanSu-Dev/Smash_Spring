# Smash_Spring

## 프로젝트 개요
- 병원에서 환자들 운동량을 측정하여 관리할 필요성 제기




## 프로젝트 사용자
1. 서버 관리자
2. 주치의
3. 운동 코디네이터
4. 환자




## 사용된 FrameWork
- Spring Framework + MyBatis(MariaDB)
